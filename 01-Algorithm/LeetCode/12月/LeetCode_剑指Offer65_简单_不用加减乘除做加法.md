### 不用加减乘除做加法

**题目描述**

```tex
写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

 
示例:

输入: a = 1, b = 1
输出: 2
 

提示：

a, b 均可能是负数或 0
结果不会溢出 32 位整数
```

**C++**

```c++
class Solution {
public:
    int add(int a, int b) {
        // 无进位时停止
        if (b == 0) {
            return a;
        }
        
        // 转换成非进位和 + 进位
        return add(a ^ b, (unsigned int)(a & b) << 1);
    }
};
```
